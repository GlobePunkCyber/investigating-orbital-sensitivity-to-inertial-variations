Experiment 004
Title: Stability Boundary Analysis

Objective:
Investigate how changes in the inertia parameter affect orbital stability.

Method:
The simulation was run using a fixed initial orbital velocity calculated for the baseline inertia factor of 1.0. The inertia factor was then varied while keeping all other parameters constant.

Observation:
Values significantly below 1.0 resulted in rapid orbital decay and impact.

Additional Analysis:
A zoomed view was generated near the baseline value to better visualize the transition between stable and unstable behavior.

Results:
Inertia Factor 1.000:
- Orbit survived the full 30-day simulation.

Inertia Factor 0.900:
- Orbit became unstable and impacted Earth before the end of the simulation.

Interpretation:
The simulation suggests a strong sensitivity of orbital stability to changes in the gravity-inertia balance when the initial orbital velocity is held constant.

Conclusion:
Large deviations from the baseline parameter produce unstable orbital behavior. Additional testing near 1.0 would be required to identify the exact transition region.
