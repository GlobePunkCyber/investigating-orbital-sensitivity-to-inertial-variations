Experiment 005
Title: Precision Threshold Analysis

Objective:
Determine how sensitive orbital position is to very small changes in the inertia parameter.

Simulation Duration:
365 Days

Tested Values:
1.000001
1.000010
1.000100
1.001000

Results:
1.000001 -> ~500 km final position error
1.000010 -> ~4,700 km final position error
1.000100 -> ~5,700 km final position error
1.001000 -> ~12,300 km final position error

Observation:
Even extremely small changes in the inertia parameter produced measurable orbital position differences after one year.

Interpretation:
The simulation demonstrates that orbital trajectories are highly sensitive to small changes in the gravity-inertia balance. Error growth appears nonlinear, with larger deviations producing rapidly increasing position errors.

Conclusion:
Long-duration orbital predictions are sensitive to small perturbations in model parameters. Tiny differences can accumulate into large positional divergences over time.
